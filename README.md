# 📊 Sales Dashboard Analysis – Data Analyst Internship (Task 4)

## 📌 Project Overview
This project focuses on designing an interactive business dashboard using Power BI.  
The objective was to analyze a sales dataset and transform raw data into meaningful insights for business stakeholders.

The dashboard provides a clear view of sales performance, trends over time, category contribution, regional distribution, and growth analysis.

---

## 🎯 Objective
To design an interactive dashboard that:

- Identifies key business KPIs
- Performs time-series analysis
- Enables dynamic filtering using slicers
- Supports data-driven decision making

---

## 🛠 Tools Used
- Power BI Desktop
- Superstore Sales Dataset (train.csv)
- Microsoft PowerPoint (for summary presentation)

---

## 📂 Dataset Information
Dataset: Superstore Sales Data  
File Used: `train.csv`

Key Columns Used:
- Order ID
- Order Date
- Sales
- Category
- Region
- Segment

---

## 📈 Key Performance Indicators (KPIs)

The following measures were created using DAX:

- **Total Sales** = SUM(Sales)
- **Total Orders** = COUNT(Order ID)
- **Average Sales per Order**
- **Year-over-Year Sales Growth %**

These KPIs provide a high-level overview of business performance.

---

## 📊 Dashboard Components

### 1️⃣ Executive Summary Cards
- Total Sales
- Total Orders
- Average Sales per Order
- Sales Growth %

### 2️⃣ Time-Series Analysis
- Sales Trend Over Time (Line Chart)
- Year-wise Sales Growth %

### 3️⃣ Category Analysis
- Sales by Category (Column Chart)

### 4️⃣ Regional Analysis
- Sales by Region (Bar Chart)

### 5️⃣ Segment Analysis
- Sales by Segment (Donut Chart)

### 6️⃣ Interactivity
- Region Slicer
- Category Slicer
- Year Filter

All visuals dynamically update based on slicer selections.
